- ==실제 내용이 표시되는 영역==
- `width`, `height`로 영역 크기 지정
	- 단, 블록의 유형마다 조절되는게 다를 수 있음
- 영역의 최대, 최소 크기를 정할 수 있음
	- `min-width`, `min-height`, `max-width`, `max-height`
		- 각 최대, 최소보다 줄어들거나 늘어나는 것을 막음
- 측정의 기준에 관해서 중요한 속성이 있음
	- `box-sizing`
		- `content-box`(기본값)
			- width, height가 가리키는 기준이 content 영역
		- `border-box` 
			- `width`, `height`가 가리키는 기준이 content + padding + border 를 포함한 크기
			- 이 경우 content영역은 자동으로 계산됨
- content영역을 벗어나면 `overflow` 현상이 발생한다
	- `overflow`
		- `visible`
			- 자르지 않고 밖을 넘어서도 표시
			- 넘치는 내용은 다른 요소 위에 겹쳐 보임
		- `hidden`
			- 넘치는 요소를 안보이게 처리(자름)
			- 스크롤바 없음
		- `scroll`
			- 넘치는 요소는 잘라내지만 스크롤바 제공
			- 넘치지 않더라도 항상 스크롤을 제공
		- `auto`
			- 넘치는 요소는 안보이게 처리
			- 필요할 때만 스크롤바 표시
	- `overflow-x`, `overflow-y`
		- 각자 가로, 세로에 대해서 어떻게 처리할 지 지정 가능
		- scroll의 단축 속성
- 배경에 관하여
	- 배경의 시작점을 정할 수 있는 속성이 존재한다 
		- ([background-clip](https://developer.mozilla.org/en-US/docs/Web/CSS/background-clip))
			- `border-box`(기본값, border부터 채움)
			- `padding-box`(padding부터 채움)
			- `content-box`(content부터 채움)
			- `text`(text부터 채움, 단 표준이 아님)
